Pour ce projet de traduction d'images Radar-vers-Optique, j'ai utilisé le dataset Kaggle 'Sentinel-1&2 Image Pairs'. Ce jeu de données pré-traité m'a permis de m'abstraire des défis complexes du pré-traitement géospatial (co-registration, alignement temporel) pour me concentrer sur l'objectif principal : la comparaison de 6 architectures SOTA de Deep Learning, de l'UNET aux modèles de Diffusion.
